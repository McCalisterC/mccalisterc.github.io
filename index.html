About Me

I'm a game developer currently studying at the University of Central Missouri. Currently, I'm learning Unity and C#. I have a few projects you can check out below!

1 Hit Wonder

This game was the winner of Texas A&M's 2024 Chillennium Game Jam's Best in Programming category!
Created in 48 hours using Unity and C#, this game is an arcade shooter reminiscent of old arcade games like Contra!

Midnight Snack

This game was the winner of UCM's 2024 Game Jam!
Made in Unity and C#, this game is a 2D platformer where you play as a racoon trying to steal all the food you can find in as little time possible! See if you can reach the top of the leaderboards for the fastest time!

A Little Bit More Than Crazy

This game was the winner of UCM's 2023 Game Jam!
Made in Unity and C#, this game is a lovecraftian JRPG where the user has to figure out what is making his school go "crazy"!

VR Zombies!

A VR game made in Unity taking aspects of the game Call Of Duty: Zombies and implementing them into a unique VR experience!

Deep Sea Ore Odyssey

A submarine mining game made in 2 weeks using Unreal Engine 5 and it's blueprint system! Explore this procedurally generated ocean and mine its minerals to upgrade your submarine so you can explore even more!
